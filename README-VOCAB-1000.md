# Vocabulary 1000 Patch

## 追加内容

- TOEIC頻出順を意識した1000語の単語帳
- 旅行・出張シーン別カテゴリ
- 50語ずつのチャンク表示
- カテゴリ別フィルタ
- 検索
- 覚えた単語数の進捗表示
- 頻出順位 `#rank` 表示

## 使い方

いつもの手順で上書き後、ローカルで確認してください。

```powershell
npm.cmd run dev
```

確認後、GitHub/Vercelに反映します。

```powershell
git add .
git commit -m "expand vocabulary to 1000 words"
git push
```
