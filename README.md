# Travel TOEIC Coach

Next.js / TypeScript / Tailwind CSS で作った、スマホ向けTOEIC学習MVPです。

## 実装済み

- ホーム画面
- 今日の10問ミニ演習
- 演習結果の localStorage 保存
- 結果画面の復習リスト
- 単語帳
- 単語の「覚えた」状態の localStorage 保存
- 学習履歴
- スマホ向け下部ナビ
- 本格アプリ風UI

## 起動方法

```bash
npm install
npm run dev
```

ブラウザで `http://localhost:3000` を開いてください。

## 主な画面

- `/` ホーム
- `/practice` 今日の10問ミニ演習
- `/vocabulary` 単語帳
- `/history` 学習履歴
