# Question Detail Analytics Patch

追加内容:

- 1問ごとの回答ログを保存
  - 問題ID
  - 文法カテゴリ
  - 旅行シーン
  - 難易度
  - 選んだ選択肢
  - 正解選択肢
  - 正誤
- 苦手分析ダッシュボードを問題別ログから集計
- 文法カテゴリ別・旅行シーン別の正答率をより正確に表示
- 最近のミスを履歴画面に表示
- 履歴一覧で「問題別ログを見る」を追加

OpenAI APIは使わないため、課金なしで動作します。

反映後、演習を1回完了すると新しい問題別ログが保存されます。
過去の履歴はそのまま残りますが、問題別ログは新しく完了した演習から記録されます。
